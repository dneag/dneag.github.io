DDK Review was a semester-long Database Management group project completed by myself and fellow classmates, Doug Rudy and Kent Hoang. The plan was to create a video game search-and-review website where users would rate games' tags rather than giving a single score. There is still much that can be improved, but we are quite happy with what we accomplished for the class.

Tools: Vue3, Tailwind CSS, PostgreSQL, Supabase, Axios, Vite.
Game data accessed through the rawg.io API.
